# Tailwind CSS：构建现代 Web 的加速器

> 无需离开 HTML 即可快速构建现代响应式网站。

## 核心特点

- **Utility-First**：通过组合基础类名（如 `flex`, `pt-4`, `text-center`）来构建完全定制的设计。
- **响应式设计**：内置简单易用的断点前缀（如 `md:`, `lg:`），让复杂布局变得简单。
- **零冗余**：构建时会自动 Purge 掉未使用的样式，最终 CSS 文件极小。
- **IDE 支持**：强大的 VS Code 插件提供完美的代码补全，极大提升效率。

## 为何推荐？

Tailwind 最大的贡献不是样式的写法，而是它定义了一套科学的设计系统约束。它能让你在不写一行独立 CSS 文件的同时，保持团队代码风格的高度一致，且完全摆脱了命名冲突的噩梦。

## 业界对比

| 特性         | Tailwind CSS     | Bootstrap                | CSS Modules / SCSS |
| :----------- | :--------------- | :----------------------- | :----------------- |
| **设计感**   | 高度自由，无味   | 模板化严重 (一看就是 BS) | 完全自定义         |
| **开发速度** | 熟练后极快       | 快 (自带组件)            | 较慢               |
| **项目规模** | 完美适配大型项目 | 基础站首选               | 标准选择           |
| **体积控制** | 最小 (扫描提取)  | 固定大小 (库)            | 灵活但易碎         |

**总结**：Tailwind 是目前构建高性能、可维护 UI 的行业标准。一旦你习惯了这种工作流，就再也不想回去写常规的 CSS 了。
