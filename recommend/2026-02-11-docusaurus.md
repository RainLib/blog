# Docusaurus：静态文档站的理想之选

> 帮助你快速构建优化的、以内容为中心的网站。

## 核心特点

- **内容定义一切**：将 Markdown 转化为现代化的 React 单页应用。
- **极致的 i18n 支持**：原生支持多语言路由和内容翻译，非常适合全球化项目。
- **丰富的插件生态**：搜索、SEO、PWA 等功能几乎只需配置即可开启。
- **React 全力驱动**：允许在 Markdown 中直接内嵌自定义 React 组件 (MDX)。

## 为何推荐？

本站点就是基于 Docusaurus 构建的。它平衡了“内容的简洁”与“功能的强大”。对于博客、技术文档或产品动态来说，它提供的搜索索引和暗黑模式支持是同类产品中最稳健的。

## 业界对比

| 特性           | Docusaurus          | Hugo                   | VitePress     |
| :------------- | :------------------ | :--------------------- | :------------ |
| **底层框架**   | React               | Go                     | Vue           |
| **构建速度**   | 一般 (Node.js)      | 极快 (Go)              | 快 (Vite)     |
| **定制化**     | 极强 (Swizzle 模式) | 较强，但需学 Hugo 语法 | 强，Vite 驱动 |
| **生态成熟度** | 极高 (Meta 维护)    | 很高                   | 正在快速成长  |

**总结**：如果你需要一个功能完备、开箱即用且高度可定制的文档/内容管理平台，Docusaurus 是目前的最佳平衡点。
