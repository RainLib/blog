# Motion Canvas：程序员的视频创作神器

> 这是一个通用的动画渲染引擎，通过 TypeScript 编写你的视频内容。

## 核心特点

- **代码即视频**：所有动画逻辑都是通过 Generator 函数编写的，版本控制极其方便。
- **实时预览**：强大的预览器，支持实时拖拽时间轴和节点检查。
- **Procedural（过程式设计）**：非常适合制作技术解释类视频，生成逻辑严丝合缝的几何图形动画。
- **现代技术栈**：基于 TypeScript 和 Vite，开发者体验极其出色。

## 为何推荐？

如果你是一个讨厌使用复杂 GUI（如 After Effects）的开发者，但又想制作高质量的技术分享动画，Motion Canvas 是目前市面上最先进的选择。它让你能用“逻辑思维”去构建“视觉艺术”。

## 业界对比

| 特性         | Motion Canvas | Manim (3Blue1Brown) | Adobe After Effects |
| :----------- | :------------ | :------------------ | :------------------ |
| **语言**     | TypeScript    | Python              | GUI / ExtendScript  |
| **实时性**   | 极佳          | 较差 (渲染较慢)     | 无需运行环境        |
| **学习曲线** | 面向前端友好  | 纯逻辑，无 UI       | 极陡峭              |
| **扩展性**   | 基于 Web 加速 | 科学计算驱动        | 插件生态丰富        |

**总结**：Motion Canvas 是目前 Web 环境下最具表现力的视频编程工具，尤其适合前端开发者进阶创作。
