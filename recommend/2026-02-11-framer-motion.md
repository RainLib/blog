# Framer Motion：React 交互动画的艺术

> 为 React 应用提供生产级的动画能力。

## 核心特点

- **声明式语法**：通过简单的 `animate` 属性即可实现复杂的补间动画。
- **手势支持**：原生支持拖拽、悬停、点击等各种交互反馈。
- **布局动画**：极少有的通过一行代码实现组件重新排列时的平滑过渡。
- **服务器端渲染支持**：完美适配 Next.js 等 SSR 框架。

## 为何推荐？

在 React 开发中，Framer Motion 几乎是实现“丝滑感”的首选。它的 API 设计极其直观，即使是动画新手也能快速上手。对于需要高度定制交互（如本站的各种磁吸和悬停效果）的项目，它是不可替代的。

## 业界对比

| 特性         | Framer Motion     | GSAP (GreenSock) | CSS Animations |
| :----------- | :---------------- | :--------------- | :------------- |
| **易用性**   | 极高 (React 友好) | 中等 (API 较多)  | 高 (原生)      |
| **性能**     | 优秀 (GPU 加速)   | 行业顶尖         | 视场景而定     |
| **功能深度** | 交互手势驱动      | 复杂时间轴驱动   | 基础路径动画   |
| **体积**     | 较轻              | 略重             | 零开销         |

**总结**：如果你正在做 React 项目，Framer Motion 是现代、美观、响应式设计的终极答案。如果你需要做超级复杂的长动画或非 React 项目，GSAP 依然是王者。
